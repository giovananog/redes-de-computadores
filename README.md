## 💻 Conteúdo

A disciplina aborda os seguintes tópicos:

1. **Arquitetura das camadas TCP/IP**
   - Associação com o Modelo OSI
   - Camada de aplicação
   - Camada de transporte
   - Camada de rede
   - Camada de enlace

2. **Planejamento e gerência de redes**
   - Segurança e autenticação
